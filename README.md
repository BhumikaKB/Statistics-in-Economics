# Statistics-in-Economics

Analyzed Indian economic datasets (GDP, income distribution, poverty) using R. Performed data cleaning, statistical analysis (growth rate, correlation, linear regression), and calculated inequality metrics such as Gini coefficient.

# Technologies Used
- Microsoft Excel
- Statistical Methods: Mean, Median, Standard Deviation, Coefficient of Variation,     Correlation, Linear Regression
- Data Visualization (Charts & Graphs)

# Data Sources
- National Accounts Statistics - Statement 1.1 (MoSPI) – GDP Data
- World Bank Poverty & Inequality Platform – Inequality and Poverty Data

# Installation
- Clone the repository
- Open the Excel workbook to explore datasets and statistical analysis.
